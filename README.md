# Visuals Library

A reusable visual explainers library rebuilt as a Next.js app for Vercel deployment.

## Tools used (and how they're linked)

- **Next.js**: frontend app framework (repo root is the app)
- **Tailwind CSS**: styling
- **GitHub**: source control
- **Vercel**: deployment target, auto-deploy on push

## How systems are linked

- GitHub repo: https://github.com/Donaid-Ngai/visuals
- Vercel: deploy from repo root
- Homepage acts as the navigation layer for visual/demo pages

## Current visuals

- `/visuals/capabilities-showcase`
- `/visuals/marven-baseball-sim`
- `/visuals/project-demos`

## Environment variables

This project currently requires no environment variables.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.
