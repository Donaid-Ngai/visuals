# Data spec

## Flow sections
### Old workflow
- Owner request arrives
- Manager inbox
- Manual forwarding to board
- Waiting / missing info
- Follow-up emails and reminders
- Documents hunted down
- Reply eventually sent

### Today workflow
- Portal access / owner request
- Board automatically notified
- Each member gets their own email
- Request logged properly
- Shared documents available
- Templated response prepared
- Faster resident reply
